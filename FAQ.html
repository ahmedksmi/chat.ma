Questions fréquentes (FAQ)

Comment puis-je créer un compte ?
Vous pouvez créer un compte en cliquant sur le bouton "S'inscrire" sur la page d'accueil et en remplissant le formulaire d'inscription.
Comment réinitialiser mon mot de passe ?
Cliquez sur "Mot de passe oublié" sur la page de connexion, puis suivez les instructions envoyées à votre email.
Puis-je modifier mon profil après inscription ?
Oui, vous pouvez modifier vos informations personnelles dans la section "Mon Profil" après vous être connecté.
Comment contacter le support ?
Vous pouvez utiliser la page "Contact" pour envoyer vos questions ou préoccupations, nous vous répondrons rapidement.
